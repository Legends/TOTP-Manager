# TOTP Manager App

WPF app for managing TOTP secrets that generate TOTP codes and QR codes.
Uses `Otp.NET` for generating TOTP codes and `QRCoder` for generating QR codes.

## Features
- Generate TOTP codes + QR codes
- CRUD TOTP secrets
- Store secrets securely

## Usage
- Click on a TOTP secret to view its details and generate a TOTP code.
- Edit a TOTP secret by double-clicking on it.
- Delete a TOTP secret by right-clicking on it and selecting "Delete".
